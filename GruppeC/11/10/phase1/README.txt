IST Beziehungen:
Ein Redakteut ist ein Nutzer aber nicht jeder Nutzer ist ein Redakteur und ein Chefredakteur ist ein Nutzer und ein Redakteur aber nicht jeder Redakteur ist Chefredakteur
Ein Album ist ein Blogeintrag
Eine Produtktrezension ist ein Blogeintrag

Parallele Beziehungen:
- zwieschen Nutzer und Kommentar:
  Nutzer kann mehrere kommentare abgeben, bearbeiten und löschen - Ein Kommentar kann nur von einem Nutzer erstellt, gelöscht oder beareitet.
- zwieschen Redakteur und Blogeintrag
  Ein Readakteur kann mehrere Blogeinträge anlegen, bearbeiten und Löschen aber ein Blogeintrag kann nur von dem Redakteur angelegt, bearbeitet und gelöscht werden. Beim Blogeintrag einlegen wird der Redakteur gespeichert, bei Blog bearbeitung wird das Änderungsdatum gespeichert.
 zwieschen Chefredakteur und Blogrintrag:
- Chefredakteur kann mehrere Blogbeiträge bearbeiten und Löschen. Ein Blogbeitrag kann von mehreren Chefredakteuren bearbeitet werden. Blogeintrag kann nur einmal gelöscht werden.
- zwieschen Readakteur und Chefredakteur bzw Nutzer und Chefredakteur:
  Ein Chefredakteur kann beliebig viele Redakteure bzw Nutzer hinzufügen und Löschen. Ein Redakteur bzw Nutzer kann nur von einem Chefredakteur hinzugefügt werden oder nur von einem oder keinem gelöscht werden.

Ein Blogbeitrag hat mindestes ein Schlagwort. Ein Schlagwort kann zu keinem oder mehreren Beiträgen gehören.

Ein Album besitzt mindestens 1 Bild und maximal 15 Bilder. Ein Bild kann in keinem oder mehreren Alben vorkommen.
